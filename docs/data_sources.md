# Public Data Sources

These sources are examples for manual research. The Python package does not call
them.

| Layer | Public tools |
|---|---|
| Narrative | CoinGecko categories, DefiLlama TVL, Artemis, Token Terminal, Kaito public views, Polymarket markets |
| Social | LunarCrush, Kaito, DexScreener trending, Galxe, Layer3, public X search |
| On-chain | Dune, Arkham, Cielo, Nansen guides, DexScreener, GeckoTerminal, Etherscan, Solscan, Helius docs |
| Catalyst | CoinMarketCal, Token Unlocks, exchange announcement pages, project blogs, governance forums |
