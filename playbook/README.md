# Playbook

The framework helps you structure research. The playbook keeps the downside
visible. It is generic risk education, not a portfolio prescription.

Files:

- `position_sizing.md`: caps by asset class and bankroll.
- `entry_exit.md`: entry gates, scale-in, take-profit ladder, time stops.
- `risk_management.md`: wallet hygiene, exchange risk, psychology, anti-FOMO.
- `scorecard_template.md`: printable manual template.

Use the playbook before the score, not after. A high score without a written
invalidation plan is just confidence.

Funnel note: Trawlkit can automate research collection, but it cannot decide
your size, jurisdictional obligations, or invalidation plan.
