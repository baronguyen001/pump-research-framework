# Case Studies

SURVIVORSHIP-BIAS CAVEAT: these are selected public examples. For every winner
listed here, many more tokens died, rugged, dumped, or stopped trading. Use the
failures file and the pattern checklist at least as much as the winners.

All multiples and windows are approximate, based on public market history at the
time of writing. This is not a list of tokens to buy.

## Index

- `01_memecoin_viral.md`: SHIB, PEPE, BONK, WIF, POPCAT.
- `02_l1_infra.md`: SOL, AVAX, MATIC, TIA, and ICP as a counter-case.
- `03_airdrop_launch.md`: JUP, ENA, HYPE, JTO, and dump counter-cases.
- `04_ai_narrative.md`: VIRTUAL, AI16Z, and the AI-agent drawdown.
- `05_failures.md`: the losers and the 2025-2026 reality check.
- `patterns.md`: 10 repeat patterns and 7 red flags.
